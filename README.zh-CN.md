# AI Formula Output Rules 中文说明

目标环境：

- `Codex App`
- `Claude Code CLI`

交付物：

- 可安装 skill：[codex-app-formula-output-rules](./codex-app-formula-output-rules)
- 可复制 CLI 文件：[claude-code-cli/CLAUDE.formula-rules.md](./claude-code-cli/CLAUDE.formula-rules.md)

安装 / 使用：

- Codex App：从本仓库路径 `codex-app-formula-output-rules` 安装 skill
- Claude Code CLI：把 `claude-code-cli/CLAUDE.formula-rules.md` 复制或合并到本地 `CLAUDE.md`

内容：

- 公式渲染规则
- prose 和 equation 的边界规则
- 公式编号与布局规则
- 一个 Codex App skill 和一份 Claude Code CLI 规则片段

语言定位：

- 这套公开规则来自中文优先的工作流
- 英文用户可以保留同样的公式边界、编号和布局规则
- 迁移到英文时，把 prose 里的中文引用词换成英文即可，例如把 `式 (1.3)` 换成 `Eq. (1.3)`
